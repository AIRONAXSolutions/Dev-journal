connecting system using SSH Via terminal

1. Install openssh server to both system

$ sudo apt update
$ sudo apt install openssh-server

2. now connecting system

$ ssh username@ip_address
ex-: ssh user@192.168.0.110

Check Status of SSH
$ sudo systemctl status ssh

Start SSH
$ sudo systemctl start ssh

Stop SSH
$ sudo systemctl stop ssh

Disable SSH
$ sudo systemctl disable ssh

Enable SSH
$ sudo systemctl disable ssh
